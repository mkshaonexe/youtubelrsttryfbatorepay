Thank you for using our Facebook App. This Privacy Policy explains how we collect, use, and protect information when our app interacts with Facebook Pages through the Meta Graph API.

By using our app, you agree to this Privacy Policy.

1. App Owner

This app is owned and operated by:
MK Shaon
(Individual Developer & Administrator)

2. Information We Collect

Our app uses the Meta Graph API to access specific Facebook Page data.
We only collect the following:

a) Page Messages

Messages sent to the connected Facebook Page

Content of the message (text only)

User ID (public Page-scoped ID provided by Facebook)

b) Page Comments

Comments made on posts of the connected Page

Comment text

Comment ID and Page-scoped User ID

c) Technical Data (Automatically Collected)

Event type (message, comment)

Timestamp

Facebook Page ID

Metadata required by the Facebook API

3. How We Use the Information

The collected data is used only for the following purposes:

a) To Generate Auto-Replies

We process message/comment text using an AI model to:

Create reply messages

Create reply comments

Provide automated customer support

b) To Improve Response Quality

We may store message/comment logs to:

Improve reply accuracy

Train custom logic (NOT for training AI models)

c) To Provide Page Admin Functions

We allow the Page owner to:

View replies

Review automated interactions

Monitor conversation logs

We DO NOT sell, share, or transfer any user data to third parties.

4. How We Store & Protect Data

If data is stored, it is stored securely in encrypted databases (e.g., Supabase / Firebase / private servers).

We use the following protections:

HTTPS encryption

Restricted server access

API token protection

No public sharing of data

All data is accessible only to the Page owner (you) and the app administrator (also you).

5. Sharing of Data

We do NOT share data with:

Advertisers

Data brokers

Third-party companies

Data is only shared with:

Google Gemini API or OpenAI API (for generating AI replies, if enabled)

These services receive only:

The message/comment text

No personal identifiers unless included by the user in the message

6. Data Retention

We retain data only as long as necessary for:

Generating replies

Improving response logic

Page insights (optional)

You may request data deletion anytime.

7. User Rights

Users interacting with your Facebook Page can:

Request deletion of their messages (> via Page admin)

Request a copy of their data

Request correction of stored information

Request that no auto-replies are used for their messages

We will honor all requests within 30 days.

8. Children’s Privacy

Our app does not target children under age 13 and does not knowingly collect information from them.

9. Permissions & Compliance

This app complies with:

Facebook Platform Policies

Meta Graph API requirements

GDPR (where applicable)

We only request permissions required for:

Reading messages

Reading comments

Sending replies

We do not access private profile data of users.
